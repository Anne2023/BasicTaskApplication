ToDo App

O ToDo App é uma aplicação simples de lista de tarefas desenvolvida em Kotlin, permitindo que os usuários criem, visualizem, atualizem e excluam tarefas a fazer.

Funcionalidades

- Adicionar uma nova tarefa
- Marcar uma tarefa como concluída
- Editar uma tarefa existente
- Excluir uma tarefa
- Visualizar todas as tarefas

Pré-requisitos

- Dispositivo Android com a versão mínima do SDK 21 (Android 5.0 Lollipop) ou superior
- Android Studio 4.0 ou superior instalado
- Conexão com a internet (para integração com serviços externos, se houver)
